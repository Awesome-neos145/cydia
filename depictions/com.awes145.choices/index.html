Choices
...is a game about picking the right choices.
This is a game where you will need to pick the right option out of 3 to win. You must input these choices 20 times. If you mess up, you will be sent to the beginning.
Each time the game starts, choices will be generated at random. There will not ever be a set of choices that will be the exact same. (unless you start the game an abhorrent amount of times, then just maybe)
This is a CLI-based game. As such, you will need a terminal emulator. MTerminal works best on iOS 6+, while MobileTerminal works best on iOS 5 and lower. Of course, those are just recommendations...
Once this package is installed, run it by typing 'choices' in the terminal emulator. You'll be presented with 3 options. To start the game, type 'play'.
Choices and its iOS port were created by awes145.
Other libraries needed to run the game were included in The LLVM Project. This inlcudes 'divsi3', 'modsi3', and 'udivsi3'. These were taken from their original forms and only compiled to armv6 and armv7 architectures. No code changes to these libraries were made.
Compiled with the cctools-port toolchain. apple-libdispatch was also used there.
